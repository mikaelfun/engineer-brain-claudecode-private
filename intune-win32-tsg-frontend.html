Metadata

Verify

Top 5 Download Failures

01
CDN Unreachable — 无法连接 swdb02-mscdn.manage.microsoft.com，防火墙/DNS 解析失败
02
China Telecom Latency — Verizon CDN 在中国电信网络下载极慢，超时或中断
03
BITS Service Failure — Background Intelligent Transfer Service 停止或损坏，回退到 HTTP 也失败
04
Proxy Blocking — 企业代理拦截 IME 到 Intune Service 的 HTTPS 连接，SSL inspection 干扰
05
EnforcementRetryCount Exhausted — 重试次数耗尽（默认 3 次），需清除注册表重置计数器
05

Detection & Installation Errors

Detection Types

  • 0 — Registry Key / Value
  • 1 — MSI Product Code
  • 2 — File / Folder Path
  • 3 — PowerShell Script

Common Pitfalls

  • 0x80070002 — Install command 里的文件名与 .intunewin 内的实际文件不匹配
  • -2016345060 — 安装后 Detection Rule 没匹配到，常因路径或版本号配错
  • 32-bit vs 64-bit PS 脚本默认 32-bit 运行，注册表路径会 redirect 到 WOW6432Node

Key Kusto Queries

1. App Install Attempts by Device

// 查询设备上 Win32 App 安装记录 ApplicationInstallAttemptsByDeviceId("{DeviceId}") | where ApplicationName contains "AppName" | project Timestamp, ApplicationName, EnforcementResult

2. Enforcement Status Events

// EventId 5767=成功, 5766=失败 DeviceManagementProvider | where EventId in (5767, 5766) | where DeviceId == "{DeviceId}" | project TimeGenerated, EventId, ResultCode

3. SideCar (IME) Service Events

// IME 服务层事件 IntuneEvent | where ApplicationName == "SideCar" | where DeviceId == "{DeviceId}" | sort by TimeGenerated desc | take 50
