# ExpressRoute Bandwidth Utilization

## Description

How to collect ExpressRoute bandwidth utilization based on average and total over a period of time.

Use cases:
- Billing Scenarios
- Customer Questions

## Dashboard

Jarvis Dashboard: Use the Jarvis link to view bandwidth utilization.

### Required Parameters
- MSEE Device Name
- Service Key
- SubInterface Name

All parameters can be found in ASC, in the ExpressRoute circuit details.

### Available Views
- **Peerings**: Shows bandwidth per peering subinterface
- **Global Reach**: Shows Global Reach bandwidth utilization
