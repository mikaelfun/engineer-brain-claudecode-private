[Docs W365 Link Device Test Peripherals](https://review.learn.microsoft.com/en-us/windows-365/link/tested-peripherals-windows-365-link?branch=release-win365-linkga)

**NOTE: Third party device drivers are not currently supported with the Windows 365 Link Device. If you need to install a device driver it is not supported.**

Microsoft engineering tests various external monitors, docks, cables, and adapters to offer the best experience on the Windows 365 Link device.
Just because a customer is using a device that is not on this list does not mean it will not work, it just means it has not been tested with the Link device.

Refer to this list as a troubleshooting step as it will be updated.

If a customer is having an issue with a particular peripheral, you might point them to this list.
