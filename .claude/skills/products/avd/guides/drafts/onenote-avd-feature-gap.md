# AVD Feature Gap - Mooncake (China)

> Source: OneNote - Mooncake POD Support Notebook
> Last updated from source: 2026-01-16
> Status: draft

GA date: September 17, 2021

## Feature Availability Matrix

| Feature | Available in Mooncake? |
|---------|----------------------|
| Android Client | No |
| iOS Client | No |
| macOS Client | No |
| Microsoft Store Client | No |
| Thin Client | No |
| Windows App | No |
| Autoscale (scaling plan) | No |
| Azure Virtual Desktop for Azure Stack HCI | No |
| Intune enrollment | Not supported |
| Azure AD MFA | Yes |
| ADFS SSO | Yes |
| Start VM on Connect | Yes |
| RDP Shortpath | Yes |
| MSIX app attach | Yes |
| Multimedia redirection | Yes |
| App attach | TBD |

## Key Notes

- Mooncake only supports Windows Desktop Client and Web Client
- Autoscale / scaling plan is NOT available - manual scaling only
- Intune management for AVD session hosts is not supported in Mooncake
- Windows App is not available; use legacy Remote Desktop client instead
