# GDPR and CCPA Data Subject Requests (DSRs)

## Process
1. Customer admin submits DSR via support case (MAC or MEM)
2. CSS collects required information
3. CSS creates ICM → assigns to SaaF
4. SaaF transfers to appropriate team
5. SaaF alerts Privacy PM for CELA review
6. Approved → SaaF/CSS responds to customer

**Deadline**: 28 days from initial request

## Required Information
- Tenant GUID (from RAVE)
- Residency (country)
- Requestor UPN/email
- DSR Type: Access, Export, or Delete
- Data Subject UPN and Object ID (single per request)
- Privacy regulation context: GDPR, CCPA, LGPD, or none
