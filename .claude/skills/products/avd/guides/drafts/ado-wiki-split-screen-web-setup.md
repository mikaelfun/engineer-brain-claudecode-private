#Setup Guide
       
User needs to navigate to windows.cloud.microsoft.com.

![Picture5.jpg](/.attachments/Picture5-62b283c9-f5bb-4fa5-a190-435bd26f324f.jpg)


Connect to VM, a new tab will open with the client.

![Picture6.jpg](/.attachments/Picture6-a0ad0620-2621-4737-99e3-0f5042e905d1.jpg)

Toggle split screen on, and click connect, make sure hardware acceleration is also on.

![Picture7.jpg](/.attachments/Picture7-8715d88f-dfe1-41a2-bcec-9490f4de2793.jpg)

The VM should now open in split screen mode.

![Picture8.jpg](/.attachments/Picture8-896aa156-73c9-4df6-9ebf-58b9c8401f7d.jpg)
