How to navigate to a resource in Azure Support Center

Instructions:
1. Open Azure Support Center from support request
2. Navigate to Resource Explorer
3. Select which view for the navigation tree: Resource Provider or Resource Group
   - Resource Provider lists resources by provider (e.g., Microsoft.Insights, Microsoft.AlertsManagement)
   - Resource Group lists resources by resource group then by provider
   Note: The subscription itself is an Azure resource
4. Search for the desired resource by populating the search dialog
5. Click on the desired resource from filtered results
6. With the desired resource selected, click the Properties tab
