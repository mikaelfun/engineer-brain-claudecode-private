# Support Boundary: Azure AI Foundry + App Insights

## Overview
Azure AI Foundry is a unified platform for building, deploying, and managing generative AI applications. It supports autoinstrumentation with Application Insights, similar to Azure Functions and API Management.

## Support Routing

| Issue Type | Team |
|---|---|
| Enabling monitoring / missing data / partial telemetry | Azure AI Foundry team |
| Alert configuration | Alerts team |
| Navigating App Insights blades / telemetry ingestion | Application Insights Monitoring team (collaboration) |

- [App Insights supported platforms](https://learn.microsoft.com/azure/azure-monitor/app/app-insights-overview#supported-platforms)
