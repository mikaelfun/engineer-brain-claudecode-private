# Support Boundary: Azure AD B2C VS Code Extension

## Overview
The Azure AD B2C extension for VS Code lets users navigate through Azure AD B2C custom policies and view Application Insights telemetry through the extension.

## Key Fact: NOT Supported by CSS
- This extension is **NOT developed by Microsoft**
- It is a community/third-party tool
- CSS does **not** support this extension
- This is explicitly called out in Microsoft public docs

## Support Path
- GitHub issues: https://github.com/azure-ad-b2c/vscode-extension/issues
- Marketplace: https://marketplace.visualstudio.com/items?itemName=AzureADB2CTools.aadb2c
