# How To: Check Log Analytics Agent Report for last heartbeat in ASC

## Instructions

1. Navigate to Azure Support Center from the case management system or via [Azure Support Center](https://aka.ms/azuresupportcenter) directly and enter the Support Request ID (aka case number).

2. Find the Log Analytics workspace that the VM is supposed to be reporting to in ASC.

3. Click on the "Agent Report" tab at the top of the workspace.

4. You can now search for the VM in this report.
   - You can view the last heartbeat recorded, version of the agent, and all solutions that the VM currently is configured for.
