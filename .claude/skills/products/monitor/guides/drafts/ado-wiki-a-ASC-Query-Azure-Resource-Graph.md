How to query Azure Resource Graph using Azure Support Center

Instructions:
1. Open Azure Support Center from support request
2. Navigate to Resource Explorer
3. Select the subscription ID under All resources
4. Select ARG Query Editor
5. Run your ARG query in the box

Note: If running a query used on a Log Analytics workspace, make sure to remove the arg(""). pattern before running the query.
