# Scenarios
---
Use the Log Analytics "Schema Onboarding" Power BI report to:
- Find which tables are associated with which Azure resource type (provider)
- Find which resource type (provider) of a given table
- Which table is available on Resource Specific mode/AzureDiagnostics/Both options
- Availability of tables in each region, including version

**Note:** This tool is only limited to Azure Resource Logs. It does not include other data types (e.g., Heartbeat).
