# Overview

Customers that have front-facing web applications running on JavaScript can use the JavaScript SDK to collect client-side telemetry from their user's browser.

This guide covers how to configure the SDK using both the snippet and the NPM package, as well as advanced concepts pertaining to the operation of the JavaScript SDK.
