#Overview
___
This detector allows the discovery of properties and settings of an Application Insights resource:
- Daily Cap and Daily Cap Reset time
- application ID (appid)
- subscription ID (subid)
- workspace ID (workspace id)
- Networking and AMPLS settings
- etc

#Considerations
___
To access this detector for an Application Insight resource that needs to be found start App Lens using any accessible, known Application Insight resource.


#Troubleshooting Guide (TSG)
___
1. Access Applens: [AppLens](https://supportability.visualstudio.com/AzureMonitor/_wiki/wikis/AzureMonitor.wiki/579985/AppLens)
1. Once the Application Insight resource loads look for the detector and select it
1. After selecting the detector the following experience will surface on the right
1. Note this data is latent as shown in the dialog at the top.
1. Note the report sections which allows for quicker navigation
