## Overview

The Access Control tab in Azure Support Center (ASC) allows you to see roles that users and groups have assigned to the Application Insights resource.

## General Usage

The Role assignments section has a drop down menu to show a list of distinct names or can be typed into and a filter applied to help locate the user easier.

This experience correlates closely to the "Access control (IAM)" tab in the Azure portal.

### When to Use
- Customer reports access/permission issues with Application Insights resources
- Need to verify RBAC role assignments on an AI component
- Checking if a user has the required role to view/modify Application Insights data
