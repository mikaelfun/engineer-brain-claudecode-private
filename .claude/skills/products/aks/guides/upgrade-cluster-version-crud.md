## Quick Troubleshooting Path

1. Check: Customer should request a CPU core quota increase via Azure Portal `[source: ado-wiki]`
2. Check: 1) Identify unsupported VM extensions on the VMSS (check AppLens detector or Azure Portal VMSS Exten `[source: ado-wiki]`
3. Check: Remove or temporarily disable resource locks on the MC_ resource group and its resources, then retry `[source: ado-wiki]`
