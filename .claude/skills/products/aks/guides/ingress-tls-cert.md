## Quick Troubleshooting Path

1. Check: Run `az aks rotate-certs -g $RESOURCE_GROUP_NAME -n $CLUSTER_NAME` to manually rotate all certificat `[source: onenote]`
2. Check: 让客户临时禁用 proxy/firewall/第三方设备的 SSL packet inspection 功能，然后重试 Start 操作 `[source: ado-wiki]`
3. Check: Inform customer that proactive/standby support is not available for AKS `[source: ado-wiki]`
