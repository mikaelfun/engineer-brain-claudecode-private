# Troubleshoot Connections Between Pods/Services in Same AKS Cluster

## Steps
1. Set up test pod
2. Verify pod running, check logs
3. Verify service endpoints match pod IPs
4. Check network policies
5. Restart kube-proxy and CoreDNS if needed
6. Check node resources
