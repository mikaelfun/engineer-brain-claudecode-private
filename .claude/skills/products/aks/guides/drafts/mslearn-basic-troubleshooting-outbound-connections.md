# Basic Troubleshooting of Outbound Connections from AKS

## Traffic Categories
1. Internal
2. Same/peered VNet
3. On-premises
4. Public via LB
5. Public via Firewall

## Checklist
1. VNet Verifier
2. DNS
3. IP reachability
4. Test from pod
5. Verify endpoint
6. External endpoints
7. Network policy
8. NSG
9. Firewall
10. Identity permissions
