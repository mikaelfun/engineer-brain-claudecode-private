## Quick Troubleshooting Path

1. Check if ADS metrics are displayed correctly in Metrics blade. If metrics display is wrong, raise an AVA and involve SME. If metrics display is correc... `[source: ado-wiki]`
2. Verify ADS MI roles via SQL query on sys.database_principals. Grant missing roles directly to ADS MI (not via security group). `[source: ado-wiki]`
3. Grant Storage Blob Data Reader for provider MI, Storage Blob Data Contributor for consumer MI on storage account IAM. `[source: ado-wiki]`
