## Quick Troubleshooting Path

1. Three options: 1) Use self-service option to add users to Data Governance role (docs: https://learn.microsoft.com/en-us/purview/data-catalog-permissio... `[source: ado-wiki]`
2. Add user to Data Governance role group via Purview settings. Data Governance role group grants Data Governance Administrator. Ref: learn.microsoft.com... `[source: ado-wiki]`
3. Assign Monitoring Reader role on Purview resource. Ref: learn.microsoft.com/en-us/azure/azure-monitor/roles-permissions-security#monitoring-reader `[source: ado-wiki]`
