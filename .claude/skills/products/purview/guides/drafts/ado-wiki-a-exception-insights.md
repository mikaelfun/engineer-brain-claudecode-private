# Exception Insights — Kusto Queries

All Exceptions that were returned during a scan.

**Input Required:** Scan Run Id

**Attached Query File:** [ExceptionInsights.txt](https://dev.azure.com/Supportability/Azure%20Purview/_wiki/wikis/Microsoft%20Purview?pagePath=%2FTools%2FKusto%20Queries%2FException%20Insights) (see wiki page for download)

## Usage Notes

- Use ASC (Azure Support Center) Exception Insights to run these queries
- Provides all exceptions encountered during a specific scan run
- Useful for diagnosing scan failures or partial scan completion
