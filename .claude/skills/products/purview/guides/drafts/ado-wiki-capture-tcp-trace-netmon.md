# Capture TCP Trace via Network Monitor

## Steps
1. Download Microsoft Network Monitor 3.4 (https://www.microsoft.com/en-us/download/details.aspx?id=4865)
2. Start Network Monitor with Administrator Privilege
3. Click New Capture, then Start
4. Reproduce the issue
5. Stop log capture
6. Save the .cap file and share
