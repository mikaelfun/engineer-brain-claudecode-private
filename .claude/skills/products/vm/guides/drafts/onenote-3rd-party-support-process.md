# 3rd Party OS Support Boundaries & Process

**Source**: Mooncake POD Support Notebook > VM > Process > 3rd Party Support Process

## Support Boundary

Microsoft only provides support for **Azure Platform and Services** when it comes to 3rd Party Vendors. It is the customer's responsibility to engage the 3rd Party Vendor.

## Three Categories

### 1. Marketplace
- 3rd party app usage and consumption on Azure
- Troubleshoot within Azure limits, then redirect to publisher for tech support within 3rd party software

### 2. Integrated Support (Ticket Exchange Platform)
- Some partners have fully integrated support via Ticket Exchange Platform (TEP)
- Partners: Red Hat, Citrix, Docker, SUSE, VMWare, SAP HANA, etc.

### 3. Linux / Commercially Viable
- Microsoft provides technical support for specific Linux and Open Source technologies on Azure
- Policy: [Support for Linux and Open Source Technology in Azure](https://support.microsoft.com/en-us/help/2941892/support-for-linux-and-open-source-technology-in-azure)
