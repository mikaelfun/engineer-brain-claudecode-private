## Information Required
Pre-requisites needed prior to troubleshooting Azure Advisor issues:

1. Email address (most important)
2. SubscriptionID
3. Time problem occurred
4. Resource Name if applicable
5. Recommendation Name if applicable

## Portal Blade Issues
If the issue relates to the Azure Advisor blade (not the recommendation blade), ask customers:

1. Are you able to reproduce this problem consistently?
2. Do you get the same result on another browser?
3. Have you tried to clear the cache?
4. Have you tried clicking on the Refresh button and still get the same result?

Run **F12** to get the console log or use **Fiddler** for network traces.
