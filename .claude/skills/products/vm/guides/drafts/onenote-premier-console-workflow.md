# Premier Console Workflow (When SAW Machine Unavailable)

> Source: MCVKB 2.29.3
> Applicability: Mooncake (21V) - fallback when SAW machine doesn't work

## Process

1. Raise 21V case using your own subscription from: https://support.azure.cn/en-us/support/support-azure/
2. Console legal consent must be sent by CSS (Microsoft) to customer - 21V only handles console setup
3. Include Teams meeting link - suggest CSS sends directly to customer
4. Do NOT add 21V engineer to premier case email directly; send meeting link separately

## Consent Template

### Chinese Version (Key Points)
- Console login only for Windows RDP / Linux SSH issues
- Customer inputs own username/password
- Remote assistance ends immediately when problem is resolved
- Subscription owner responsible for account operations
- 21Vianet follows customer requirements and platform expert guidance
- Owner has right to audit/record all remote access operations

### English Version (Key Points)
- Application is for specific subscription ID only
- Applicant must be subscription owner, enterprise employee, or authorized 3rd party
- 21Vianet has right to verify 3rd party applications
- Operations strictly per customer/expert group requirements
- Must comply with relevant laws and regulations
- Privacy: information kept confidential, used only to provide assistance
