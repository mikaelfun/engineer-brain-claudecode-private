1. Create a dummy/fake policy targeting impacted devices using the opposite assignment type: if all policies use Device groups, add a User group assignme `[来源: OneNote]`
2. 1) Enable Microsoft Account Sign-in Assistant service; 2) Enable Network Connection Broker (ncbservice); 3) Verify device ticket exists in registry; 4 `[来源: OneNote]`
3. 1. 确认设备 Windows 版本满足最低要求（Win11 23H2 需 22631.2506+KB5031455）；2. 检查 C:\Program Files\Microsoft Device Inventory Agent\Logs（InventoryAdaptor.log 和 Intune `[来源: ADO Wiki]`
4. 1. 检查设备注册表是否 ToastEnabled=0（被禁用）；2. 启用 Toast Notification（取消 GPO 限制或调整 Intune 策略）；3. 确认 WNS 服务正常运行；4. 注意：On-Demand Remediation 不需要脚本分配到用户/设备组，但必须启用 WN `[来源: ADO Wiki]`
5. Intune only delivers drivers. If driver causes issues, engage TA/TL to discuss with Windows DND team or redirect customer to OEM vendor directly `[来源: ADO Wiki]`
