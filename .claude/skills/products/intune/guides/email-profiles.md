## 快速排查路径

1. **Samsung KNOX: EAS email profile error 0x87D1FDE8 remediation failed.**
   → Deselect Samsung Notes sync option. Wait up to 24h. `[mslearn, 🟢 8.0]`

2. **Users cannot send images from auto-configured email accounts.**
   → Enable the setting in email profile Properties > Settings. `[mslearn, 🟢 8.0]`

3. **iOS email profile: no password prompt after password change. No email sent/received.**
   → Enable OAuth in email profile. Ensure email service supports OAuth first. `[mslearn, 🟢 8.0]`

4. **When trying to configure the Microsoft Intune On-Premise Exchange connector, error: 'Configuration Failed - The Microsoft Intune Exchange Connector version is not supported. Mobile device managemen...**
   → Download and install the latest version of the Intune on-premise Exchange connector from the Exchange Access blade. `[contentidea-kb, 🔵 7.5]`

5. **Alert: 'The Service to Service Connector Failed to Retrieve ACS Token' when using Intune O365 Exchange Connector.**
   → Wait 24 hours (may be maintenance). If not resolved: ensure Global Admin who set up the Service Connector has both Intune and O365 Exchange licenses. Verify certificate and ACS registration. `[contentidea-kb, 🔵 7.5]`
