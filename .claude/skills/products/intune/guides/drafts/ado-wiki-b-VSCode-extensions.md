# VSCode Extensions for Intune Support Engineers

VSCode extensions recommended as replacements for Notepad++ (NP++, now blocked).

## Encode/Decode Base64

**Extension**: MIME Tool  
Use: Review certificates or tokens. Select text → `Ctrl+Shift+P` → search "Base" → select decode/encode command.

## Find In File

**Extension**: Find In File  
Use: Find all instances of text in a file. `Ctrl+Shift+P` → "Find all..." → "Find All In File: String"

## Highlight Log Lines

**Extension**: Log Highlight  
Use: Highlight lines by keyword/regex. Works with plain text or regex patterns. Filters panel in Explorer sidebar.

## Find in Workspace

**Extension**: Easy Search  
Use: Search across all files in workspace. Default key: `Shift+f` (recommend rebinding to avoid conflict with native 'f' capitalization).

Note: Easy Search does not support Find & Replace — use native VSCode search for that.
