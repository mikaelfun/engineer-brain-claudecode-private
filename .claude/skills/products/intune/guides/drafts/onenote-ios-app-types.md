# iOS App Types in Intune

Source: OneNote iOS TSG

## App Types
1. **Store Apps** - From Apple App Store, auto-update
2. **LOB Apps** - Custom .ipa files, manual update
3. **Web Apps** - URL shortcuts on home screen
4. **Built-in Apps** - Pre-installed, managed as-is
5. **VPP Apps** - Purchased via ABM, centralized licensing, auto-update
6. **Managed App Configuration** - Apps supporting managed config settings

## Key Differences
- VPP: No Apple ID needed with device licensing
- LOB: Admin uploads .ipa and manages updates
- Store: Linked from App Store, requires Apple ID on non-supervised devices
