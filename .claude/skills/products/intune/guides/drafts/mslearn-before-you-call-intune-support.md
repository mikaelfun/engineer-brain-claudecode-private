# Before You Call Intune Support - Preparation Checklist

## Required Information
- Tenant name (e.g., contoso.onmicrosoft.com)
- MDM authority
- Issue history: Has this feature/device/process ever worked?
- Company Portal app version (latest?)

## Issue Description Best Practices
- Be as specific as possible
- State ultimate goal (what you are trying to accomplish)
- Specify area: device management, app management, enrollment, or administration
- Include device types, OS, affected apps, observed vs expected behavior

## Visual Documentation
- Include screenshots at ticket creation time
- Use Windows Steps Recorder (search "steps" in Start menu) to capture exact reproduction steps
- All uploaded files are saved securely

## Device Management Checks
- Follow all steps in the device management documentation for your device type

## App Protection Policy Checks
- Is the application updated to latest version?
- APP without enrollment or enrolled devices?
- Which applications are affected?
- What device type?
