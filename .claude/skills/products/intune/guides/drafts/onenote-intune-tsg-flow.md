# Intune TSG Flow (Decision Tree)

## First: Collect Basic Info
1. Intune Device ID
2. User ID
3. Policy/App ID (if policy or app deployment issue)

## Branch by Scenario

| Scenario | TSG Topic |
|----------|-----------|
| App deployment (general) | App deployment |
| Win32 app / PowerShell script | Win32 deployment / IME troubleshooting |
| Office 365 ProPlus | Intune O365 ProPlus deployment |
| macOS agent issues | Mac Agent / SideCar workflow |
| Shell scripts | Shell TS |
| Group membership questions | Device and User Effective Group |
| License issues | Check Intune license |
| Device configuration profiles | Device Configuration |
| SCEP/certificate issues | NDES TSG |
| Compliance policy issues | Check device compliance |
| App Protection / App Configuration | APP/ACP TSG |
| Enrollment issues | Device Enrollment |
