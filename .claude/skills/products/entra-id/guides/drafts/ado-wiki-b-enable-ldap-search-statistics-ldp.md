# Enable LDAP Search Statistics in LDP

## Summary

This guide provides instructions on how to enable Search Statistics for LDAP within LDP by adjusting search options and configuring controls.

## Steps

1. Open LDP tool
2. Modify the **search options** to **Extended**
3. Click on **Controls**
4. In the Controls section, check the control for **Search Stats**
5. The **BaseDN** (Base Distinguished Name) and **Filter** will reflect the LDAP queries you are investigating
