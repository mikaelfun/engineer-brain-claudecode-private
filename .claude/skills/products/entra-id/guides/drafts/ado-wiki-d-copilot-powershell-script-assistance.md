# Feature overview

**DRAFT: This is an unreleased product. Documentation still in development.**

# Case handling

This feature is supported by **M365 Identity**. Issues with specific PowerShell cmdlets and cmdlet output is supported by the community that owns that specific cmdlets. Support for scripting is limited, please refer to the following documents for official CSS Scripting support guidance:

- [Policy: Custom Code and Scripts](https://internal.evergreen.microsoft.com/en-us/topic/5f21b5e3-5122-6eab-5abd-789db31ef57a)
- [Procedure: Custom Code and Scripts](https://internal.evergreen.microsoft.com/en-us/topic/ef3296ef-1480-ae0f-1741-7b838d32fd36)

# Limitations and known issues

Copilot PowerShell Assistance will only use the following modules:

| Description | Links |
|---|---|
| Graph PowerShell | microsoftgraph-docs-powershell (graph-powershell-1.0, docs-conceptual, scripts) |
| Office PowerShell | office-docs-powershell (officewebapps, exchange, spmt, teams, whiteboard) |
| PowerShell Core Cmdlets | PowerShell-Docs (reference/7.4, reference/7.5, docs-conceptual) |
| SharePoint PowerShell Docs | OfficeDocs-SharePoint-PowerShell (sharepoint-ps, docs-conceptual) |
