# Collect Network Trace (Fiddler) with SaRA Tool

> Source: OneNote - M365 Identity IDO / Common operation_TSG

## 场景

需要收集 Office/M365 应用的 Fiddler 网络抓包时，使用 SaRA (Support and Recovery Assistant) 工具简化操作。

## 步骤

1. 从 Assist 365 启动 Diagnostics → 选择 Network Capture → 点击 Start
2. 完成后获取包含 unique key 的 diagnostic summary
3. 指导客户按照以下文档操作：[如何运行 Fiddler 跟踪](https://learn.microsoft.com/zh-cn/microsoft-365/troubleshoot/diagnostic-logs/run-fiddler-trace)

## 注意

- SaRA 工具内置 Fiddler 功能，客户无需单独安装 Fiddler
- 适用于 Windows 平台的 Office/M365 应用网络问题排查
