# Copilot Settings

Comprehensive reference of all Copilot settings available in the M365 Admin Center Copilot Control System page.

| Name | Description | Information/Notes |
|------|-------------|-------------------|
| Web search for Microsoft 365 Copilot and Microsoft 365 Copilot Chat | Provide Copilot with additional functionality by allowing access to web search. | Shortcut to create a cloud policy in the Microsoft 365 Apps admin center. When you enable the Allow web search in Copilot policy setting, Copilot can reference web content to improve the quality of its responses. |
| Recommendations for Microsoft 365 Copilot licensing | Choose whether admins can view recommendations for Copilot licenses. | Microsoft provides licensing recommendations based on Microsoft 365 apps activity. By default, this setting is enabled. Requires License Administrator or User Administrator role. |
| Pin Microsoft 365 Copilot Chat | Choose whether users have Microsoft 365 Copilot Chat pinned across their experiences. | Pins Copilot Chat to the navigation bar in Teams, Outlook, and the Microsoft 365 Copilot app. By default, Copilot Chat might be pinned for users with a Microsoft 365 Copilot license. |
| Pin Microsoft 365 Copilot apps to the Windows taskbar | Choose whether to pin the Microsoft 365 Copilot app and optional companion apps to the user's Windows taskbar. | Off by default. |
| People Skills in Microsoft 365 Copilot | Manage People Skills in Microsoft 365 and Copilot experiences. | Shortcut to Settings > Viva > Data management > People Skills. AI-powered service delivering personalized skills-based experiences. |
| Opal (Frontier) | Choose which users in your organization can access Opal (Frontier). | Enterprise automation experience using Computer-Using Agents (CUA) to automate multi-step tasks. |
| Microsoft Copilot for Security | Go to Microsoft Copilot for Security to manage these settings. | Direct shortcut to the Security Copilot portal. |
| Microsoft 365 Copilot self-service purchases | Control product trials and purchases to enable for end users. | Three options: Allow (try or buy), Allow trials only (free trial, no auto-convert to paid), Do not allow. |
| Microsoft 365 Copilot in admin centers | Choose whether admins can access Microsoft 365 Copilot within admin centers. | Affects users with admin roles in M365 admin center, Exchange admin center, SharePoint admin center, and Teams admin center. Allowed by default. |
| Data security and compliance | Manage how Copilot references documents and sites in the Microsoft Purview portal. | Shortcut to the Microsoft Purview portal. |
| Copilot video generation | Manage whether users can create generative AI videos with Copilot. | Uses responsible AI to help block harmful content during upload or creation. |
| Copilot pay-as-you-go billing | Set up pay-as-you-go billing for Copilot and related agents. | Opens the Copilot Billing & usage page. Switch to Pay-as-you-go services tab and select Microsoft 365 Copilot Chat to configure. |
| Copilot in Teams meetings | Go to the Microsoft Teams admin center to change how Copilot is used in meetings. | Shortcut to the Microsoft Teams admin center. |
| Copilot in Power Platform and Dynamics 365 | Manage more settings related to Copilot and agents in Power Platform and Dynamics 365. | Shortcut to the Power Platform admin center. |
| Copilot in Edge | Copilot in Edge gives your organization AI-powered chat for work with enterprise data protection. | Shortcut to Settings > Microsoft Edge > Configuration policies. |
| Copilot in Bing, Edge, and Windows | Manage how your organization uses Copilot, your AI-powered chat for the web. | For M365 subscription users: Microsoft 365 Copilot Chat with enterprise data protection. For personal accounts: consumer Microsoft Copilot. |
| Copilot image generation | Manage whether users can generate images with Copilot. | When allowed, users can ask Copilot to create, design, and edit images. If not allowed, responds with stock or brand images. |
| Copilot Frontier | Enable exclusive early access program in your organization. | By default, no users have access to Frontier features. |
| Copilot diagnostic logs | Send diagnostic logs for Copilot on behalf of individual users who are having issues. | Temporarily overrides any user level feedback policy. |
| Copilot Custom Dictionary | Manage custom dictionaries for your organization. | Upload custom dictionaries to enhance Copilot's word recognition. Helps with proper nouns, technical jargon, and transcription in Teams. |
| Copilot AI disclaimer | Choose how prominently the AI disclaimer is shown to users in Microsoft 365 Copilot. | Appears in Word, Excel, PowerPoint, Outlook, OneNote, and Microsoft 365 Copilot app. Does NOT appear in SharePoint, OneDrive, Whiteboard, and Forms. |
| AI providers operating as Microsoft subprocessors | Manage external AI providers that are Microsoft subprocessors. | Anthropic onboarded as Microsoft subprocessor. Previous opt-in option under Anthropic's separate terms is deprecated. |
| AI providers for other large language models | Manage third-party model providers running outside of Microsoft. | Configure in the Microsoft 365 admin center. |
