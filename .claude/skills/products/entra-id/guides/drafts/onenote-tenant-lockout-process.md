# Tenant Lockout / Last Admin Forgot Password (21V Process)

## Scenarios

### Scenario 1: Last Admin Forgot Password
- Handled by **21V Commerce team**
- Customer calls T0 hotline: **+86 400-089-0365**
- Or creates commerce case via https://support.azure.cn/zh-cn/support/support-azure/ → "Support Type - Subscriptions Management"

### Scenario 2: Tenant Lockout (by CA or Legal/Compliance)
- Same handling as Scenario 1 → **21V Commerce team**
- Same contact channels as above

### Scenario 3: Disable Global Admin MFA
- Open case to **21V Technical team** (VM + SCIM POD)
- Refer to [[11.27 [Process][MFA] How to disable MFA]]

## Important Notes
- **None** of the above scenarios have 24x7 support
- Since it involves engaging legal and other teams, **ETA > 1 week**
- Set customer expectations accordingly
