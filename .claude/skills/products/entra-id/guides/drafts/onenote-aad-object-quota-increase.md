# Increase AAD Object Quota in Mooncake

## Background
- **Free edition**: max 50,000 objects per directory
- **With verified domain**: quota extended to **300,000 objects**
- **Beyond 300K**: requires MSODS team assistance via IcM

## Steps to Increase Quota

1. Collect from customer:
   - Initial Domain Name
   - Tenant ID
   - Target Quota

2. Submit IcM ticket to **MSODS/QuotaIncreases** team
   - Template: https://portal.microsofticm.com/imp/v3/incidents/create?tmpl=j2K351
