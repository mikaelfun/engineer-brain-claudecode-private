# Enable LSA Policy Debug Logging

## Summary

Guide for enabling Local Security Authority (LSA) policy debugging logging for AD performance cases related to LSA policy requests.

## Steps

1. **Refer to internal KB for debug flags and settings:**
   - Navigate to: [LSA Policy Debug Logging Flags and Options](https://internal.evergreen.microsoft.com/en-us/topic/adperf-tools-lsa-policy-debug-logging-flags-and-options-7703843e-9662-5545-0a10-448ea2dceeb4)

2. **Configure Debugging Flags:**
   - Open the debugging tool and set the appropriate flags for LSA policy debugging
   - Ensure you select flags relevant to your performance case
