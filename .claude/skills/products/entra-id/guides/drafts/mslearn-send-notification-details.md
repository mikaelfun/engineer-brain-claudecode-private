# How to Send Notification Details to Microsoft Support

**Source**: [Microsoft Learn](https://learn.microsoft.com/en-us/troubleshoot/entra/entra-id/app-integration/send-notification-details)

## Steps

1. Click Notifications bell icon (top-right of Azure portal)
2. Select any notification in Error state (red ! icon)
3. Review Notification Details

## Key Fields to Collect
- Title, Description, Notification ID
- Client Request ID, Timestamp UTC
- Internal Transaction ID, UPN
- Tenant ID, User Object ID
- Status, Object ID, Details
- Copy error JSON (errorCode + localizedErrorDetails)

## Sharing with Support
- Use "Copy error" button to copy all details
- Share ALL listed details for fastest resolution
