## Get all template specs in a subscription
Use [[JARVIS] Get resources from provider](https://jarvis-west.dc.ad.msft.net/22260025?genevatraceguid=9c3b980c-0fd1-4a10-a478-11087ede2944) action.
![image.png](/.attachments/image-1518123f-82f0-464a-95dd-265a3567f83b.png)

<br>

## Get a specific template specs resource or version in a subscription
Use [[JARVIS] Get resource from URI](https://jarvis-west.dc.ad.msft.net/3AD62B49?genevatraceguid=2c0bd0d1-79a9-4491-b599-c7860f3d5910) action.
![image.png](/.attachments/image-46884ae8-1513-4050-a388-7fa178767e02.png)

This will give the full version details, including the template itself.
