# Configure Network Security Groups with Windows Admin Center

## Overview
NSGs define inbound/outbound traffic rules. WAC provides a graphical interface to manage.

## Key Concepts
- Rule Definition: allow/deny based on source, destination, port, protocol.
- Policy Scope: VM, subnet, or VNET level.
- WAC Integration: user-friendly UI.
- Logging: NSG flow logs for audit and troubleshooting.

## Internal Guidance
- Enforce least privilege access.
- Apply rules consistently with templates.
- Document exceptions for audit.
