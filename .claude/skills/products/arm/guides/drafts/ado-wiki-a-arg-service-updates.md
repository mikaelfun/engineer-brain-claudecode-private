## 11/13/2023
### Log Analytics can now consume data from ARG
Log Analytics now offers the `arg("")` function, which lets users consume data from ARG in a Log Analytics query — either to join ARG data with LA tables, or to use additional LA functionality (such as alerting) over ARG data.

More information: [Announcing the Public Preview of Alerting on Azure Resource Graph](https://techcommunity.microsoft.com/t5/azure-governance-and-management/announcing-the-public-preview-of-alerting-on-azure-resource/ba-p/3980771)
