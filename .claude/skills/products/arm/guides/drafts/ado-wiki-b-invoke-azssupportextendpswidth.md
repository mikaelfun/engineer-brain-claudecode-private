# CSSTools Invoke-AzsSupportExtendPSWidth

# Synopsis

Function will be run to modify the ERCS PS width size to get full output - default is 350

# Examples

## Example 1

```powershell
Invoke-AzsSupportExtendPEPPSOutput
```

## Example 2

```powershell
Invoke-AzsSupportExtendPEPPSOutput -BufferSize 500
```
