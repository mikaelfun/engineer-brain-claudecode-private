## 2. STANDARD DASHBOARD BUTTON PATTERNS

PATTERN 1: PRIMARY ACTION BUTTONS
- Size: 12px font, padding 9px 18-22px, radius 13px
- Colors: var(--accent-blue/green/red) solid background
- Text: Bold, uppercase, 0.3px letter-spacing
- Icon: w-4 h-4, always paired with text
- Shadow: Inset rgba(0,0,0,0.06) 0 1px 3px 0
- Used for: Start, Cancel, New Cases

PATTERN 2: SECONDARY MINI BUTTONS
- Size: 10px font, px-2.5 py-1, radius 13px
- Colors: var(--accent-color-dim) background with color text
- Text: Semibold, uppercase, 0.2px letter-spacing
- Icon: w-3 h-3, spin conditionally
- Hover: brightness-110
- Disabled: opacity-40
- Used for: Case refresh, process actions

PATTERN 3: ICON-ONLY BUTTONS
- Size: p-1 padding, no set font (icon only)
- Colors: Transparent, color-driven icon
- Icon: w-3.5 h-3.5
- Hover: bg-[var(--bg-hover)]
- States: Color change (green/gray based on state)
- Used for: Edit, delete, enable/disable, run

PATTERN 4: TEXT BUTTONS WITH SMALL ICON
- Size: px-2 py-1, text-xs, r-4
- Colors: Transparent background, text-driven color
- Icon: w-3 h-3 with text
- Hover: smooth color transition
- Used for: Edit draft, copy, auxiliary actions

PATTERN 5: CONFIRMATION BUTTONS
- Size: px-1.5 py-0.5, text-[10px], rounded
- Colors: Confirm=var(--accent-red), Cancel=text-tertiary
- Layout: Horizontal group, gap-1
- Used for: Delete confirmations

## 3. UNIVERSAL DESIGN LANGUAGE

CSS CLASSES:
- Layout: inline-flex items-center gap-1/2, flex-col
- Typography: font-bold/semibold, uppercase
- Interactions: transition-all, disabled:opacity-40
- Hover: hover:brightness-110 or hover:bg-hover

COLOR SCHEME:
- Blue (var(--accent-blue)) = Primary/start
- Green (var(--accent-green)) = Positive/enable/new
- Red (var(--accent-red)) = Danger/cancel/delete
- Dim variants = Secondary/muted actions

STATES:
- Loading: Loader2 spinner replaces icon
- Disabled: opacity 0.6, cursor not-allowed
- Hover: brightness-110 or background change

## 6. KEY FINDINGS

NO CENTRALIZED BUTTON COMPONENT - inline style + className pattern
SEMANTIC COLOR CODING - blue/green/red consistently used
ICON ALWAYS WITH TEXT - except pure-icon toolbars
LOADING = LOADER2 SPINNER - replaces action icon
THREE VISUAL TIERS - large primary, medium secondary, small icon
CONSISTENT SIZING - 12px primary, 10px secondary, xs tertiary
LUCIDE ICONS ONLY - Plus, Play, Square, Refresh, Pencil, Trash2, etc
