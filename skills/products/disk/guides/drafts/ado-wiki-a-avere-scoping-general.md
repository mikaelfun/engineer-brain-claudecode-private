# Avere vFXT/FXT — Scoping Questions: General

Use these questions at the start of any Avere FXT or vFXT support case to establish baseline context.

## Initial Scoping Questions

> - Cluster Name(s)?
> - Description of the problem?
> - Is this a new issue or an existing issue?
> - When did the problem start?
> - Is the problem ongoing, or has it already cleared?
> - What is the timeframe(s) for this problem?

## Change History

> - Has this configuration worked in the past, or is this a new configuration?
> - Have there been any recent changes to the Avere configuration?
> - Have there been any recent changes to the client configurations?
> - Have there been any recent changes to the network or related infrastructure configurations?
