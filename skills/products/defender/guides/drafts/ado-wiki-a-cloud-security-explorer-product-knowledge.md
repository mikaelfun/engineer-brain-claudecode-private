# Cloud Security Explorer — Product Knowledge

[[_TOC_]]

## Overview

Defender for Cloud's contextual security capabilities helps security teams reduce the risk of impactful breaches. Defender for Cloud uses environmental context to perform a risk assessment of security issues, identifying the biggest security risks and distinguishing them from less risky issues.

The Cloud Security Explorer enables proactive identification of security risks in cloud environments by running graph-based queries on the **Cloud Security Graph** (Defender for Cloud's context engine). It allows prioritization of security team concerns while accounting for the organization's specific context and conventions.

## Availability

| Aspect | Details |
|--|--|

| **Required Roles & Permissions** | Security Reader, Security Admin, Reader, Contributor, Owner |
| **Clouds** | Azure, Amazon Web Services (AWS) |
