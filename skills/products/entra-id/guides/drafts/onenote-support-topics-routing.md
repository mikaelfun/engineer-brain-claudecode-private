# Entra ID Support Topic Routing Reference

> Source: OneNote - Mooncake POD Support Notebook / Azure AD / Overview / Support Topics

## Key Notes
- ADFS server related issues: customer must raise Premier case to ADFS team
- Most Entra ID topics route to **POD Azure VM**
- Developer-focused topics (App Development, GraphAPI, Managed Identities integration) route to **POD Azure PaaS Developer**
