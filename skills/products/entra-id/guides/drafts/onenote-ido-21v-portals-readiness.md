# 21V M365/IDO Portal URLs & Readiness

## Overview
21Vianet 环境下 M365 各组件的管理门户 URL 和 IDO 团队的 readiness 矩阵。

## 21V Portal URLs

| Component | URL |
|-----------|-----|
| Entra ID (AAD) | https://entra.microsoftonline.cn/ |
| M365 Compliance | https://compliance.microsoftonline.cn/ |
| M365 Security | https://security.microsoftonline.cn/ |
| Exchange Admin | https://admin.exchange.microsoftonline.cn/ |
| Exchange ECP | https://partner.outlook.cn/ecp |

## IDO Readiness Components

| Category | Components |
|----------|-----------|
| Identity | AAD Billing/Licenses, M365 Application (token generation), M365 PowerShell Module (token generation) |
| Compliance | Audit Events, eDiscovery, Information Protection (AIP), Data Lifecycle Management (retention policy), DLP |
| Security | O365 Service Health |
| Security-Exchange EOP | Exchange Online Overview, Mail Flow, Anti-phishing/Anti-spoofing, Anti-malware, Anti-Spam |
