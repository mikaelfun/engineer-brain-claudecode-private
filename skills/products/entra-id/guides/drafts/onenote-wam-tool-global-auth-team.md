# WAM Tool by Global Auth Team

**Source**: OneNote — M365 Identity IDO / Common operation_TSG

## Overview

WAMToolScript and WAMTroubleshooter web app are diagnostic tools built by the global authentication team for troubleshooting WAM (Web Account Manager) issues on Windows.

## Tools

### WAMToolScript (v1.0.1)
- Download: Available from CSSChinaAADAuth ADO repository (`.attachments/WAMTool/WAMToolScript_v1.0.1.zip`)
- Purpose: Local script for WAM diagnostics and state inspection

### WAMTroubleshooter Web App
- URL: https://wamtsapp.z7.web.core.windows.net/
- Purpose: Web-based WAM log analysis and troubleshooting

## Usage

- Run WAMToolScript locally to collect WAM state and logs
- Upload results to WAMTroubleshooter web app for automated analysis
- Tool provides visual diagnostics of WAM token flows, cache state, and error analysis
