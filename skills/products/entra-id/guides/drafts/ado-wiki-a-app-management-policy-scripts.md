# App Management Policy Scripts

This wiki contains helpful scripts for App Management policies that are either in public documents or created by colleagues.

## IMPORTANT - Custom Code and Scripts Policy

Make sure to review the Custom Code and Scripts [Policy](https://internal.evergreen.microsoft.com/en-us/topic/5f21b5e3-5122-6eab-5abd-789db31ef57a) and [Procedure](https://internal.evergreen.microsoft.com/en-us/topic/ef3296ef-1480-ae0f-1741-7b838d32fd36) documentation before assisting customers with codes and scripts!

When you provide customers with custom-built assets such as PowerShell scripts, you should also provide them with a license and disclaimer. This is to let them know that the assets you're providing are not Microsoft products and you are not responsible for how they use them. **Customers should test these assets before using them with their production data.**
