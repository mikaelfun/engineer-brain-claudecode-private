# Ember Log Collection Tool

## Overview
Ember is a blazing-fast log collection and download tool for Entra ID troubleshooting.

## Tool Download
- Download from: https://ember-update.azurewebsites.net/api/HttpServe

## Usage

### Log Collection
1. Open Ember application
2. Configure log collection parameters
3. Start collection

### Log Download
1. Enter the incident ID
2. You will be redirected to browser for SSO authentication
3. Return to the tool to download logs
4. Alternatively, find log files by following the directory path shown in the tool
