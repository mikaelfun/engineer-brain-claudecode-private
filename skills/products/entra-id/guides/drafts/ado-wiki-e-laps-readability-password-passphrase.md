**Summary:**

Windows Local Administrator Password Solution (LAPS) has introduced a new Password Complexity option that omits the most confusing characters to improve password readability. Additionally, LAPS has introduced a passphrase feature aimed at enhancing security while improving usability. This feature allows IT administrators to generate less complex yet secure passwords or passphrases that are easier to read, remember, and type compared to traditional complex passwords. This feature includes customizable policy settings, predefined word lists, and full integration with Windows Server Active Directory (AD) and Microsoft Entra ID.

[[_TOC_]]

#### Technical details

1. **Passphrase generation**
   - The new feature enables the creation of passphrases such as "EatYummyCaramelCandy," which are more user-friendly compared to traditional passwords like "V3r_b4tim#963?".
   - Passphrases are generated by concatenating a specified number of words randomly selected from predefined word lists. The first letter of each word is capitalized to enhance readability.

2. **Policy settings**

   **Old Windows LAPS policy** | **Windows Server 2025/24H2 Windows LAPS policy**
   -- | --
   ![image.png](/.attachments/image-4dd4052f-4754-42cf-9116-f165360ba1f0.png)  |  ![image.png](/.attachments/image-08c4028f-7060-408d-bdb1-8499edbe82ed.png)

   - **Password complexity** - The policy setting's registry value can be configured to select one of four new values (5-8):
     - **Value 5** - `Large letters + small letters + numbers (improved readability)` equivalent to PasswordComplexity=4 (`Large letters + small letters + numbers + specials`) with the following modifications made to improve readability and avoid confusion (For example, no more mistakenly typing the letters O or o instead of the number 0 and vice versa):
       - Removes the letters 'I', 'O', 'Q', 'l', and 'o'
       - Removes the numbers '0' and '1'
       - Removes the symbols ',', '.', '&', '{', '}', '[', ']', '(', ')', and ';'
       - Adds the symbols ':', '=', '?', and '*'
     - **Value 6-8** - Three different passphrase word lists, all included in Windows without requiring a separate download:

       | Password complexity GPO option | Description | Number of words in list | Remarks |
       | --- | --- | --- | --- |
       | 6 | Passphrase (long words) | 7776 | Words length is between 3 to 9 letters |
       | 7 | Passphrase (short words) | 1276 | Maximum word length is 5 letters |
       | 8 | Passphrase (short words with unique prefixes) | 1276 | 1) None of 1276 words in the wordlist starts with the same 3 characters<br>2) Actually, the words in the list are not necessarily short, maximum word length is 10 letters |

   - **Passphrase length** - A new policy setting's registry value that controls the number of words used in the passphrase. Administrators can specify the number of words, which are then randomly selected from the chosen word list.

3. **Word lists**
   - The passphrase word lists used in the three new PasswordComplexity passphrase settings are sourced from the Electronic Frontier Foundation's article, "Deep Dive: EFF's New Wordlists for Random Passphrases".
   - These word lists are licensed under the CC-BY-3.0 Attribution license and are available for download. However, customization of the built-in word lists or the use of customer-configured word lists is not allowed.

4. **Integration and support**
   - This feature fully supports backing up passwords to either Windows Server Active Directory (AD) or Microsoft Entra ID.
   - The "Large letters + small letters + numbers (improved readability)" option and the passphrase support are only available in Windows 11 24H2, Windows Server 2025, and later releases. It is not required to deploy Windows Server 2025 domain controllers to use this new setting.

5. **Windows LAPS tab new font**
   - The Windows LAPS tab in the Active Directory Users and Computers snap-in now displayed in a new font that enhances its readability when shown in plain text:

#### Example of a passphrase

An example of a passphrase created with six words taken from the "Long words" list is:
`SkiingProduceIdentifyStarlitOctaneDistress`<br>
This approach ensures that the passphrases are not only secure but also user-friendly, making it easier for you to manage and remember your credentials.<br>
For more detailed information, you can refer to the official documentation on Windows LAPS passphrase word lists and policy settings.
