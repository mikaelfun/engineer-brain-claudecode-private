# Extend Azure Active Directory Trial License

If you would like to extend your lab trial license for Azure Active Directory which is near to expiry, then send a request to **AAD Premium Demo Request** at aadpdemo@microsoft.com.

CC your manager and abigail.brown@microsoft.com.

Include the following in your request:
- Tenant name
- Tenant ID

> **Note:** Submit at least **30 days before license expiry** so your request is processed on time.
