# Collect Network Trace (Fiddler) with SaRA Tool

**Source**: OneNote — M365 Identity IDO / Common operation_TSG

## Overview

Use Microsoft SaRA (Support and Recovery Assistant) tool with Assist 365 diagnostics to collect Fiddler network traces for Office/authentication troubleshooting.

## Steps

1. From **Assist 365**, start Diagnostics → Select **Network Capture** → Click **Start**
2. Once done, a diagnostic summary is generated with a **unique key**
3. Ask customer to collect the trace following: [How to run Fiddler trace after installing SaRA](https://learn.microsoft.com/en-us/microsoft-365/troubleshoot/diagnostic-logs/run-fiddler-trace)

## Notes

- SaRA simplifies Fiddler trace collection for non-technical users
- The unique diagnostic key links the trace back to the support case
