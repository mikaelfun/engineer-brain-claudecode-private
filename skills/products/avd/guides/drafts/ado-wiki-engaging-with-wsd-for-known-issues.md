# Engaging with WSD for Known Issues

Windows Servicing and Delivery (WSD) Comms team are the route to communicate with customers and partners on important issues affecting W365/AVD. They cover comms to:

- Windows release health message center and known issue pages
- Microsoft 365 admin center
- Windows Update History Release Notes
- Windows Knowledge Base (KB) articles on support.microsoft.com

## Process

1. WSD landing page for initiating known issue requests: https://microsoft.sharepoint.com/teams/wxd/SitePages/Documenting-Windows-Known-Issues.aspx
2. Complete the intake form: https://aka.ms/WSDKnownIssueIntakeForm
3. For support/guidance, contact WSD Comms team: wsdcommski@microsoft.com
