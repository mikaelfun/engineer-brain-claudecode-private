You can use this registry setting to enable the logging of client certificate validation failures, which are events generated by the Schannel security support provider. Logging of client certificate validation failures is a secure channel event, and is not enabled on the host by default.<br>

[Enable Schannel event logging in Windows and Windows Server](https://learn.microsoft.com/en-US/troubleshoot/developer/webapps/iis/health-diagnostic-performance/enable-schannel-event-logging)

Note: Usually capturing "Auth scripts" (commonly used by the DS team) is the best approach to gather Schannel data.
