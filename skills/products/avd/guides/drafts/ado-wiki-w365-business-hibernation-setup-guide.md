# Connecting to the Business Cloud PC

## Steps

1. Select Start using Windows 365 Business.
2. Select Start using Windows 365 Cloud PCs.
3. Select Open in browser.
4. Select Connect and sign in.

## What CSS should verify

- Cloud PC successfully provisions.
- User can connect via browser.
- After 1 hour of inactivity, session resumes on reconnect.
