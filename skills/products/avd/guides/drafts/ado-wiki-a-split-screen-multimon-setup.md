# Split Screen & Multimon for Web Browser - Setup Guide

## Steps

1. User needs to navigate to **windows.cloud.microsoft.com**

2. Connect to VM - a new tab will open with the client

3. Toggle **split screen on**, and click connect. Make sure **hardware acceleration** is also on.

4. The VM should now open in split screen mode.

## Notes

- Hardware acceleration must be enabled for split screen to work
- The feature is accessed through the web browser client at windows.cloud.microsoft.com
- See also: related troubleshooting guide for when split screen/multimon is not working (avd-ado-wiki-014)
