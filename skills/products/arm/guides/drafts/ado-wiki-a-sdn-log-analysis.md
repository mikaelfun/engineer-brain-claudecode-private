# SDN Log Analysis

With SDN Logs collected you can upload to a Helen workspace and then analyse from a SAW device using the **Azure Stack Hub SDN Resources v2 Dashboard**.

This dashboard was created for Azure Stack Hub but can display most of the information from an Azure Local SDN Diagnostic collection.

> **Note**: There are currently plans in place to create an SDN Dashboard dedicated to Azure Local.
