## ARM Deployment Troubleshooting

### 1. Find the impacted deployment
Identify the specific deployment that failed or is experiencing issues.

### 2. Look for the error message (if available)
See if the error message is something you can assist with directly.

### 3. Identify if ARM or the RP is responsible for the issue
Determine whether the failure originates from ARM itself or from the downstream Resource Provider (RP).

If the issue is coming from the RP, engage the CSS team for the impacted resource.
