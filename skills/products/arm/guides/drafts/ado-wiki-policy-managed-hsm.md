Azure Key Vault Managed HSM (Hardware Security Module) is a fully managed, highly available, single-tenant, standards-compliant cloud service using FIPS 140-2 Level 3 validated HSMs.

MHSM uses Azure Policy to enforce and audit settings on MHSM data plane objects.

## Support ownership

| Component | Ownership |
|-|-|
| Policy resources (definition, initiative, assignment) | Azure Policy |
| Policy workers | Azure Policy |
| Policy Dataplane service API | Azure Policy |
| MHSM | Key Vault team |
