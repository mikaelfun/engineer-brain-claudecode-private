# Azure Local Nexus Glossary

## Nexus-specific
- **CAPI** = Cluster API: manages Kubernetes workloads
- **CR** = [Platform Cluster Kubernetes] Custom Resource
- **HIS** = Hybrid Identity Service: service used as part of Arc agent communication to Azure cloud
- **NNI** = Network to Network Interconnect (networkToNetworkInterconnects)

## Technical Terms
- **gNMI / GNMI** = gRPC Network Management Interface. See [gNMI specification](https://openconfig.net/docs/gnmi/gnmi-specification/#grpc-network-management-interface-gnmi)
- **MSEE** = Microsoft Enterprise Edge. Cisco ASR1K router providing connectivity between customer premises and Azure (ExpressRoute architecture)

## Cases and IcM Jargon
- **IDC** = India Development Center
- **JRT** = Joint Recovery Test
  - **Joint** = involves multiple domains (infrastructure, platform, network functions)
  - **Recovery** = simulates full-stack outage/failure (thermal, power, site down)
  - **Test** = validates restoration sequence, stability, data integrity
- **ORT** = Operational Readiness Test (validation step to ensure hardware system is fully functional and ready after provisioning, firmware updates, or reboots)
