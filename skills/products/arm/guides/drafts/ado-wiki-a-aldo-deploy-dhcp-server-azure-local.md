**Overview**
How to set up a DHCP server in Azure Local and deploy an Azure Local VM via DHCP in the Azure portal.

**How to setup a DHCP server**
1. Install the DHCP role in DC, and unauthorize in the DHCP console
2. Configure the DHCP Scope
3. DHCP scope option

**How to deploy an Azure Local VM via DHCP in the Azure portal**
1. After DHCP server is installed and configured, create an Azure Arc virtual machine in the Azure Portal
2. Create Virtual Network

**From DHCP server, you can find the DHCP address in DHCP server**

**The tenant VM obtains its IP address via DHCP.**
