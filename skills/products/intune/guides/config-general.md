## 快速排查路径
1. Collect F12 trace, extract client-request-id. Query IntuneEvent by ActivityId to find exception, then CMService by ActivityId to confirm service timeo `[来源: OneNote]`
2. Option 1: Deploy PowerShell script via Intune (run as SYSTEM) to set registry HKLM\Software\Policies\Microsoft\Windows NT\Printers\PointAndPrint\Restr `[来源: OneNote]`
3. MDMDiagnosticlog や Event Viewer (Event ID 2097: Windows Firewall With Advanced Security/Firewall) でルール適用を確認。Monitoring > Firewall セクションで実際のルール状態を検証する `[来源: OneNote]`
4. 1) Remove or exclude password policy from configuration/compliance profiles targeting kiosk devices; 2) Delete the EAS registry path HKLM\SYSTEM\Curre `[来源: OneNote]`
5. Verify connectivity: curl https://enterpriseregistration.windows.net/ -D - from admin cmd; check proxy/firewall rules; if Reset-LapsPassword -Verbose  `[来源: OneNote]`
