# Case Buddy for DPs
To facilitate CaseBuddy access for your Delivery Partners, kindly distribute the outlined steps to them. Additionally, please be informed that no installation or administrative privileges are necessary on their AVDs.

1. On the CE&S AVD desktop, right click, select New and then Shortcut
2. On the shortcut box type this string : `\\emeacat.europe.corp.microsoft.com\CaseBuddy\Bootstrap\BootStrap.exe`
3. Click Next and on the shortcut name input "CaseBuddy 64" and click Finish – you should now have the below icon which you can use to launch Case Buddy and set it up

Additional information about case buddy can be found in this Wiki section: [Intune Wiki](https://supportability.visualstudio.com/Intune/_wiki/wikis/Intune/1524316/CaseBuddy)

## PCMS
The Proactive Case Management Service (PCMS) is a key component of the Quality strategy, to increase the consistency in delivering great customer experiences.
PCMS is a solution designed to help Team Managers and TAs with identifying, prioritizing, and recovering open at-risk cases from poor customer experience.

For AIPD PCMS Information, please access the PDMS Dashboard:
