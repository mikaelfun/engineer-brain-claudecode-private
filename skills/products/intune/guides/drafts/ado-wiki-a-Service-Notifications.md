# Service Notifications — Reference Portals

## Summary

Reference for Microsoft service and product incident web portals (internal and external) for checking service notifications related to Intune or related services.

**Sample Use Case:** When you need to check details on a Message Center notification ID (e.g. MC1234567), use Iridias Message Center to get more information.

---

## Portal Reference Table

| Web Portal | Description |
|---|---|
| [Iridias Message Center](https://iridias.microsoft.com/messagecenter) | Provides Service Notifications information across M365 and D365. **Internal website.** |
| [Lynx](https://iridias.microsoft.com/messagecenter) | Provides data insights into a Customer at the Tenant level, enabling proactive action planning. **Internal website.** |
| [Microsoft service health status](https://status.cloud.microsoft/) | Overview status info on M365 Admin Center, Power Platform Admin Center, and Microsoft Azure, with links to each service's detailed status portal. **Public website.** |
| [Azure status](https://azure.status.microsoft/status) | Provides Azure products and services status (overall and region specific). **Public website.** |
