# Backlog Review Process

The backlog review process is a systematic approach to managing and prioritizing outstanding issues or cases within SCIM management tools and support environments. It involves regularly assessing and updating the backlog to ensure timely resolution and alignment with organizational goals.

## Review Process
- All reviews conducted in PCMS/Case Buddy using customized templates
- For cases aged 90+ days: common group chat where every engineer must share daily updates
- Each engineer's bin should include a case summary for all cases

## Benchmark

| Case Age bucket | 1-30 | 31-60 | 61-120 | 121-180 | 181 and above |
|-----------------|------|-------|--------|---------|---------------|
| Benchmark% | 75% | 20% | 4% | 0.9% | 0.1% |

- For BC team: PTAs not expected to get involved as early
- For Premier team: engagement should be quicker in early stages
- PTAs can adjust reviewers for each age group based on available resources
- Reference report: Backlog Overview Report - Power BI

## Age Group Reviews

### Age Group: 0-7
**Reviewers:** Tier 2 Engineers from the DP side (BC & Premier business); PTAs (Premier business)

| Accountability | Responsible Role | Outcome |
|---|---|---|
| Determine issue category | Reviewer | Is it Intune or not? |
| Conduct quality review (FQR) | Reviewer | Quality review conducted |
| Ensure proper case scoping | Reviewer | All cases to be scoped |
| Update SAP | Reviewer | SAP updated |
| Handle in-scope cases | Reviewer | Cases handled/transferred |
| Identify and raise collabs | Reviewer | Collaborations raised |
| Evaluate case note quality | Reviewer | Case note quality and pinned |
| Monitor case idleness | Reviewer | Case not idle |
| Check case status | Reviewer | Right case status on ticket |
| Adopt the right workflow | Reviewer | Apply the tech workflow per Wiki |
| Oversee criteria compliance | Process Technical Advisor | All cases reviewed |

### Age Group: >7 to <14
**Reviewers:** Technical Managers (TMs), Team Leads (TLs), Partner Technical Advisors (PTAs)

| Accountability | Responsible Role | Outcome |
|---|---|---|
| Post the case on SME channels | Reviewer | Case posted on SME channels |
| Initiate L1 collaboration | Reviewer | L1 collaboration initiated |
| Discuss the case in triages | CO | Case discussed in triages |
| Evaluate case note quality | Reviewer | Case note updated and pinned |
| Monitor case idleness | Reviewer | Case not idle |
| Check case status | Reviewer | Right case status on ticket |
| Monitor compliance | PTA | All cases reviewed |
| Ensure clarity in case notes | PTA, TL, TM | Clarity ensured |

### Age Group: >=14 to <=30
**Reviewers:** TMs, TLs, SME, PTAs

| Accountability | Responsible Role | Outcome |
|---|---|---|
| Conduct SME review | Reviewer | Review the post over ICD |
| Initiate L1 collaboration | Reviewer | L1 collaboration initiated |
| Discuss the case in triages | Reviewer | Case discussed in triages |
| Evaluate case note quality | Reviewer | Case note updated and pinned |
| Monitor case idleness | Reviewer | Case not idle |
| Check case status | Reviewer | Right case status |
| Report on 30+ cases status | TL/TM | 30+ cases status reported |
| Monitor compliance | PTA | All cases reviewed |

### Age Group: >=30 to <=60
**Reviewers:** TMs, TLs, PTAs

| Accountability | Responsible Role | Outcome |
|---|---|---|
| Review and identify leaders | TL/TM | Leaders identified |
| Conduct SME review | SME | SME engaged |
| Conduct PTA review | PTA | PTA review conducted |
| Conduct SEE review | SEE | SEE engaged over collab |
| Initiate L2 collaboration | Reviewer | L2 collaboration initiated |
| Evaluate case note quality | Reviewer | Case note quality and pinned |
| Monitor case idleness | Reviewer | Case not idle |
| Check case status | Reviewer | Right case status |
| Report on 60+ cases status | TL/TM | 60+ cases status reported |

### Age Group: 60+
**Reviewers:** TMs, TLs, PTAs

| Accountability | Responsible Role | Outcome |
|---|---|---|
| Review and identify leaders | TL/TM | Leaders identified |
| Conduct SME review | SME | SME review conducted |
| Conduct PTA review | PTA | PTA review conducted |
| Conduct SEE review | SEE | SEE review / picked up the case |
| Initiate L2 collaboration | Reviewer | L2 collaboration or ICM initiated |
| Report on 90+ cases status | TL/TM | 90+ cases status reported |
| Evaluate case note quality | Reviewer | Case note quality and pinned |
| Monitor case idleness | Reviewer | Case not idle |
| Perform necessary actions | Reviewer | Necessary actions performed |
| Report 90+ status to SDMs | TL/TM | 90+ status reported to SDMs |
