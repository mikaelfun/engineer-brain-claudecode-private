# Premier Reporting — How to Access EMEA/APAC/NOAM Metrics

Navigate to [aka.ms/cesbihub](https://aka.ms/cesbihub)

- From the left screen side select **Reporting Catalog**
- Then select **CSS**
- Then select **A&I and DTP**

From the new screen, select the **A&I and DTP All Up** tab and from it select **Insights+ V2**.

To use filters and to view EMEA Premier Data press on **"Full Report Filtering Page"** and use the following filter values:

- **Fiscal Date**
- **Planning Category**: SCIM Intune Advanced
- **Service Offering**: Important — exclude **Advance Support For Partners** & **Premier Support For Partners** under Premier Support, keep the rest as is.
- **Customer Geo**: Select appropriate region (EMEA / Asia / United States)
- **Employee Site**: Include all SEs working from CNX Side

Once you configure all the above filters, press on the **back button** to view metrics for the selected region.

Similar steps apply for:
- **APAC**: Select *Asia* in Customer GEO
- **NOAM**: Select *United States* in Customer GEO

**IPD Filters on IPD Created Table** are also available on the full report page.
