# iOS MDM Error Codes Reference

Source: OneNote iOS TSG

Comprehensive error code table for iOS MDM profile installation, enrollment, VPN, certificate, and app management operations.

See source page for full error code mapping (error code number -> description).
Key ranges: 1000-1010 (profile parsing), 4000-4031 (profile install), 5000-5018 (passcode), 9000-9006 (certificate), 12000-12081 (MDM), 15000-15005 (VPN), 22000-22013 (SCEP/PKI).
