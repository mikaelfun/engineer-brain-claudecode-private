# Daily Triage Case Discussion

Engineers can discuss cases (Technical actions or case management) in Daily Triage which will be led by either SEE or TA.

- ROTA for Daily Triage leader will be shared in Excel Sheet (Teams)

## Notes

- Every new case discussed will be added to OneNote — only TAs and SEEs will have read-write access to add newly cases and update next actions discussed with engineer accordingly.
- If engineer needs further assistance and discussed case in triage after x days, Triage Lead (by SEE or TA) will be able to see last suggested actions to engineer and follow-up on its results then share next actions accordingly.
