## 快速排查路径
1. Inform customer: (1) HTTP OPTIONS cannot be disabled — required for portal functionality. (2) Cipher `[来源: ado-wiki]`
2. 1) Delete the existing Bastion deployment from the VNet. 2) Redeploy Bastion with Premium SKU and en `[来源: ado-wiki]`
3. 1. Verify AADSSHLogin or AADSSHLoginForLinux extension is listed under VM > Extensions + Application `[来源: ado-wiki]`
4. Verify Linux distro is in supported list: Common Base Linux Mariner, CentOS, Debian, openSUSE, RHEL, `[来源: ado-wiki]`
5. Generate a new SAS token from the storage container: 1. Right-click container > Generate SAS; 2. Per `[来源: ado-wiki]`
