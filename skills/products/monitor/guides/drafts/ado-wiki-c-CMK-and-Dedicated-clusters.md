# Scenario
---
The purpose of this workflow is to guide you to troubleshoot issues related to Log Analytics dedicated clusters and customer-managed-key (CMK).

##Minimum information needed
* Subscription ID 
* Time window of when the issue occurred
* Issue type:
     * Cannot create / remove the dedicated cluster resource
     * Cannot update the cluster resource with the key identifier
     * Cannot associate / disassociate a workspace with the cluster resource
     * Other
* Error message (if applicable)
* Workspace name or resourceID
* Dedicated cluster name or resourceID
