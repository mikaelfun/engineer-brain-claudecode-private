# Support Boundary: Azure Batch + App Insights

## Overview
Azure Batch is a cloud-based job scheduling service for large-scale parallel and HPC applications. It provides Application Insights integration.

## Integration
- Azure Batch provides means of enabling Application Insights: [Monitor with Application Insights](https://learn.microsoft.com/azure/batch/monitor-application-insights)
- Prerequisites: [Batch + App Insights prerequisites](https://learn.microsoft.com/azure/batch/monitor-application-insights#prerequisites)

## Before External Engagement
- Verify the customer's App Insights configuration aligns with the guidelines in the prerequisites article

## SAP Routing
- `Azure/Batch Service/Monitoring and Diagnostics`

## Data Collection
- Precise ask to the team
- Resource URIs of the product/service involved
- Application Insights resource URI
