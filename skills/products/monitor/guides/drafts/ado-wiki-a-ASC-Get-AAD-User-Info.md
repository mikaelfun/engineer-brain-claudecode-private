How to get user information for Azure Active Directory (AAD) user in Azure Support Center

Instructions:
1. Open Azure Support Center from support request
2. Click on Tenant Explorer in the top navigation menu
3. Click on User in the left navigation menu
4. In the Search for cloud user tab, provide user information (email or UPN) and click Run

Reviewing the results:
If the user is found successfully, details about the user will be returned.
