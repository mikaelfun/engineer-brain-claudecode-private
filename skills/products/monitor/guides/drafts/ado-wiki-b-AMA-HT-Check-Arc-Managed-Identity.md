## Azure Portal
```
Azure Portal > Azure Arc > Machines > Select Arc Machine > Overview > JSON View > identity
```

## Azure Support Center
```
Azure Support Center > Resource Explorer > Arc Machine > Properties > Identity > Type contains "SystemAssigned"
```
