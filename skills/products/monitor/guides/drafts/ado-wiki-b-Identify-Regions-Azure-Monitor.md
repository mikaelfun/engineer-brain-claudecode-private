# Introduction

Customers will sometimes open a support request to ask about if or when a particular Azure Monitor feature will be available in a specific region. This article provides details that you can share with a customer.

> **Internal Note**: There is an internal version of this information on [Azure Global](https://global.azure.com/product-availability/roadmap). Use the Search property (top left) to look for the service (e.g., Azure Monitor) and hover over the location to get details on the current status and plans.

# Instructions

1. Open a browser and navigate to [Product available by region](https://azure.microsoft.com/global-infrastructure/services/).
2. Under **Products**, type **Azure Monitor** and then select the **Azure Monitor** option that is presented.
3. Click the drop-down under **Regions** to select your desired region.
4. Review the **Table Key** to see which regions have the desired feature generally available, in preview or planned for future.

> **Note**: If a customer wants to have an Azure Monitor service added to a region where it is not currently available, this would be a feature request. See [How to advise a customer to submit a feature or design change request for Azure Monitor](guides/drafts/ado-wiki-b-Advise-Customer-Submit-Feature-DCR.md) for details.
