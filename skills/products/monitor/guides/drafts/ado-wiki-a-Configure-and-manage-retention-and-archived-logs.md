# Scenario
The purpose of this workflow is to guide you to troubleshoot issues with configuring and managing the retention of archived logs.

## Minimum information needed
* Subscription ID
* Time window of when the issue occurred
* Issue type: Configuring workspace data retention, Table data retention and archive policy, Purging retained data, Other
* Error message (if applicable)
* Workspace name or URI, region
* Name(s) of the table(s)
