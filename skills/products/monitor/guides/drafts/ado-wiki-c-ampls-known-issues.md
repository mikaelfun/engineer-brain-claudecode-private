## AzMon Wiki Known Issue Database
This will document current and on-going issues that are on-going that customers will commonly encounter.

- [AMPLS Known Issues](https://supportability.visualstudio.com/AzureMonitor/_search?text=Tags%3A%22AMPLS%22&type=workitem&pageSize=100&filters=Projects%7BAzureMonitor%7DWork%20Item%20Types%7BKnown%20Issue%7DStates%7BPublished%7D)

- Azure Monitor Private Link (AMPLS) issues are tagged solely with "AMPLS" at this time.

- Review [Root Cause an AMPLS Issue](/AMPLS-\(Azure-Monitor-Private-Link-Scope\)/How%2DTo/Azure-Monitor-Private-Link-Scope-General-Resources/Root-Cause-an-AMPLS-Issue) for root cause selection options

Some of the Known issues here could be linked to actual workitems the Product Team has created to address the issue. Usually, the iteration path to fix the issue is a codename. See [Semester Planning Break down](https://microsoft.sharepoint.com/teams/AzureCommunity/SitePages/Semester-Planning.aspx) to decipher the codename and when that release would release.

> **Note**: To search current AMPLS known issues, use the Supportability workitem search linked above filtered by Tag "AMPLS", Type "Known Issue", State "Published".
