# Instructions
---
This article will cover how to print an Azure Workbook.

# Process
1. Load up the Azure Workbook, at the end of the name click on the '...' to locate the options menu:
![image.png](/.attachments/image-1e18075e-eaf4-4d2d-bb42-23a6867145f4.png)

2. Select 'Print content'.

3. The print page for the browser will appear, adjust settings as necessary to fit the items on the page and then print.

**Note:** This will only print the view shown on the screen, if the customer has a tabbed Workbook and needs to print out each page they will have to click through the tabs and print separately.
