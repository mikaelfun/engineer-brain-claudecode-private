# Recreate RDP Listener

## Summary

How to recreate the RDP Listener when it is broken or misconfigured.

## Prerequisites

- Windows VMs only
- Helpful to have another working VM with same OS to export RDP listener configuration

## Instructions

For detailed instructions, follow the [Windows UEX Wiki document](https://supportability.visualstudio.com/WindowsUserExperience/_wiki/wikis/WindowsUserExperience/725033/Recreate-RDP-listener).
