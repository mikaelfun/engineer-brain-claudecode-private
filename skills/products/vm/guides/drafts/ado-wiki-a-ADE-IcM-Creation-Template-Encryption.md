**Create an ICM through the Azure Support Center (ASC):**

To engage the EEE/PG teams, we will need to create an IcM and this will be done from within Azure Support Center (ASC).

1. After accessing the ASC case page, go to the right top of the page and select **Escalate Case**.

2. Select All --> Search for the ADE template: **h3s3mb** --> Select it and click on Next.

3. Fill in the required fields and click on Submit.
