# Azure Serial Console - Escalation Procedures

## Overview
Azure Serial Console escalation contacts and procedures for Mooncake support engineers.

## Escalation Channels

### 1. Teams Channel
- **SME - Serial Console teams channel**: For initial questions and guidance
- Channel: CONN - Serial Console (AVA)

### 2. ICM Escalation
- For feature-level escalation, file an ICM to the **Azure Serial Console team**
- ICM template: https://portal.microsofticm.com/imp/v3/incidents/cri?tmpl=I3y3i2

### 3. Bugs / Feedback
- Email: azserialhelp@microsoft.com

## Notes
- Serial Console availability in Mooncake may differ from Global Azure
- For password reset via Serial Console on Ubuntu, see known issue vm-onenote-210
- For guest OS dump capture without Serial Console, see guide: onenote-capture-guest-os-dump.md
