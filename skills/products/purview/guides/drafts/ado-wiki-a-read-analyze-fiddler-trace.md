# Read Fiddler Trace

1. Launch Fiddler Tool.
2. Click on File and uncheck Capture Traffic
3. Click File again and select Import Sessions…
4. Select HTTPArchive and click Next
5. Choose the .har file and click Open to analyze the traces.
