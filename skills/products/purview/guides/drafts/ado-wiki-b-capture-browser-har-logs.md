# Capture Browser (HAR) Logs

### Refer to the steps below to capture browser trace:

1. Launch Edge or Chrome and press **F12** on the keyboard to open Developer Tools.
2. Click on **Network** tab and select "**Preserve Log**" (kindly refer the screenshot)
3. Reproduce the issue.
4. Right click any of the frames under Name tab and save all the network log as HAR file.
5. Please attach this HAR file to expedite the investigation
