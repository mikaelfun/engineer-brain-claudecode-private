# Governance Skill Reference Guide

## Skill Group Definitions

Reference for AI skill groups in CRMGlobal (Machine Learning Tab).

### Key Skill Groups and MyHorizon Roadmaps

| # | Skill Group | Description |
|---|-------------|-------------|
| 1 | SkG: Catalog Management | Unified Catalog: Governance Domains, Data Products, Enterprise Glossary |
| 2 | SkG: Health Management | Health Actions, Controls, Reports, Data Quality Services |
| 3 | SkG: Billing | Pay-As-You-Go billing, new billing meters |
| 4 | SkG: Administration | Deployments, Tiers, Upgrades |
| 5 | SkG: Discovery | Search capabilities of Unified Catalog |
| 6 | SkG: Data Sources | Data source connections |
| 7 | SkG: Data Map | Data Map operations |
| 8 | SkG: Scan | Scan operations |
| 9 | SkG: Workflow | Workflow management |
| 10 | SkG: Data Estate Insights | Insights and reporting |
| 11 | SkG: UX | User experience |

Each skill group has L100/L200/L300 training paths in MyHorizon.
