 #SLA Alert for all Purview cases with CRMGlobal Bot.
it will send you notification when a case close to SLA no matter it get assigned to anyone or not assigned yet.

Commands in order:
- "what notifications are available"
- "Service Request"
- "Expiring SLA Alerts (6+hits count)" or any other SLA related options with different alert count
- "governance"
- "bot"

![image.png](/.attachments/image-283b0e06-73fb-4b3e-8042-15cc169f5d75.png)> Screenshot displaying a conversation between a user and a CRMAI bot focusing on setting up SLA alerts. The bot asks the user which Support Pod they are referring to, to which the user responds with "governance". The bot then presents notification channel options, and the user selects "Bot". Following the selection, a successful subscription message for "SLA Alerts" is shown, specifying that alerts will be generated when a case SLA is expiring and continues to do so 6 or more times, approximately 15 minutes before expiring.
