When creating an escalation, please get this information:

- Description of the expected vs actual behavior
- A [network trace](https://dev.azure.com/ASIM-Security/Compliance/_wiki/wikis/Welcome/9071/How-to-Capture-a-network-trace) of the issue in OWA
- The affected label
- The email file saved as an .msg or .eml
