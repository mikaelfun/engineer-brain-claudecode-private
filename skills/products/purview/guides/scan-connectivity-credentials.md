## Quick Troubleshooting Path

1. 1) Verify credentials are correct. 2) Confirm data source is accessible from Azure IR. 3) Check NSG and firewall rules to allow traffic from the IR. `[source: ado-wiki]`
2. Follow steps to delete the bad node that cannot connect to managed identity (see linked guide: Pod cannot connect to managed identity) `[source: ado-wiki]`
3. Verify Salesforce app has OAuth enabled and correct IP restrictions. Check 2 KV secrets: Secret 1 = user password (append security token if SHIR IP no... `[source: ado-wiki]`
