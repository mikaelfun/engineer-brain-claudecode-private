## Summary

将 TestLab.tsx 的视觉设计从当前朴素风格升级为 Design A "Mission Control" 风格：玻璃态卡片、发光特效、大号 Pipeline Hero、智能折叠推理面板。同时支持深色/浅色主题切换。

## Context

TestLab 是测试自动化框架的 Dashboard 页面，用于展示 Supervisor + Stage Worker 的实时状态。当前 V2 重构已完成功能层面（状态文件拆分、SSE 推送、新 API 端点），但 UI 视觉还是最基础的卡片布局，用户反馈"不太好看"。

已通过 3 个 HTML 设计方案对比，用户选定了 Design A "Mission Control" 风格。参考文件：`dashboard/testlab-v2-design-A.html`。

- [ ] Connector 线条有完成/流动/待定三种状态样式
