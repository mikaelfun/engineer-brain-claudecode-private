# Specification: WebUI Issue Implement Auto-Scroll

**Track ID:** implement-auto-scroll_20260322
**Type:** Feature
**Created:** 2026-03-22
**Status:** Draft

## Summary

点击 Issues 页面的 Implement 按钮后，页面自动平滑滚动到该 issue 卡片位置，方便用户实时追踪实现进度。

## Context

Issues 页面的 Issue 列表可能很长（分组 + 分页），点击 Implement 后 ImplementPanel 在 issue 行下方展开，但页面不会自动滚动到该位置，用户需要手动寻找。

## User Story

As a developer using the Issues page, I want the page to auto-scroll to the issue I just clicked Implement on, so that I can immediately track the implementation progress without manually searching.

## Acceptance Criteria

- [ ] 点击 Implement 按钮后，页面平滑滚动到该 issue 行
- [ ] 滚动后 issue 行和 ImplementPanel 都可见
- [ ] 不影响其他 issue 行的展开/折叠行为

## Dependencies

- Issues.tsx — IssueRow 组件（当前无 DOM ref）
- ImplementPanel.tsx — implement 进度面板

## Out of Scope

- 滚动到 Track Creation 或 Verify 面板（仅 Implement）
- 跨页自动跳转（如果 issue 在其他分页）

## Technical Notes

- 在 IssueRow 上添加 React ref
- 点击 Implement 后调用 `ref.current.scrollIntoView({ behavior: 'smooth', block: 'start' })`
- 使用 setTimeout 微延迟确保 ImplementPanel 已挂载后再滚动
