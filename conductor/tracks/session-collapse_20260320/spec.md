# Specification: Session Active Panel Collapsible

**Track ID:** session-collapse_20260320
**Type:** Bug
**Created:** 2026-03-20
**Status:** Draft

## Summary

CaseAIPanel 中 "Session Active" 区域及其附属内容（SSE 消息列表、Chat 输入框、Quick Actions）无法折叠，占据大量页面空间，影响查看 case 其他信息。

## Context

CaseAIPanel 是 CaseDetail 页面顶部的 AI 操作面板。当 session active 时，会显示：
1. Session Active 指示条（第 312-323 行）
2. SSE 消息列表（第 332-340 行，`max-h-60`）
3. Chat 输入框（第 342-361 行）
4. Quick Actions 按钮组（第 363-385 行）

这些内容当前是无条件显示的静态 `<div>`，没有折叠/展开机制。

## Problem Description

**Steps to Reproduce:**
1. 打开 Dashboard → 进入某个 Case
2. 执行 Full Process 或任何操作使 session 变为 active
3. 尝试折叠 "Session Active" 区域 → 无法折叠

**Expected:** Session Active 区域可以折叠/展开，折叠后只显示一个紧凑的状态指示条
**Actual:** Session Active 区域始终展开，占据大量空间

## Acceptance Criteria

- [ ] Session Active 区域可以点击折叠/展开
- [ ] 折叠状态下只显示紧凑的指示条（Session Active 标签 + session ID 摘要 + 展开按钮）
- [ ] 展开状态下显示完整内容（消息列表 + Chat 输入 + Quick Actions）
- [ ] 默认状态为展开（保持当前行为）
- [ ] 折叠/展开有平滑过渡动画

## Dependencies

None

## Out of Scope

- 改变 Session Active 区域的功能逻辑
- 修改 session history（已有 `<details>` 折叠）
- 添加最小化到角落等高级交互

## Technical Notes

**Fix approach:** 将 Session Active 指示条改为可点击的折叠触发器，用 `useState` 控制下方内容的显示/隐藏。

```tsx
const [isSessionExpanded, setIsSessionExpanded] = useState(true)

{/* Session Active — clickable header */}
{isLiveRunning && (
  <div
    onClick={() => setIsSessionExpanded(!isSessionExpanded)}
    className="flex items-center gap-2 mb-3 px-2 py-1.5 bg-purple-50 rounded-lg cursor-pointer"
  >
    <span className="w-1.5 h-1.5 bg-purple-500 rounded-full animate-pulse" />
    <span className="text-xs text-purple-700 font-medium">Session Active</span>
    <ChevronDown className={`w-3 h-3 text-purple-400 ml-auto transition-transform ${isSessionExpanded ? '' : '-rotate-90'}`} />
  </div>
)}

{/* Collapsible content */}
{isSessionExpanded && (
  <>
    {/* messages, chat input, quick actions */}
  </>
)}
```

`ChevronDown` 已在文件的 import 中，可直接使用。
