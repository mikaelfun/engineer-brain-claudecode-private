# Specification: AR Case 显示 Main Case 的 IR 状态

**Track ID:** ar-ir-display_20260404
**Type:** Feature
**Created:** 2026-04-04
**Status:** Draft

## Summary

AR（Additional Resource）类型 Case 的 IR SLA 状态显示 unknown，因为 `fetch-all-data.ps1` 在 AR 模式下跳过了 IR 检查。应改为对 Main Case 查询 IR 状态，并保存到 AR Case 的 meta 中展示。

## Problem Description

- **根因：** `fetch-all-data.ps1` 第 204 行 `if ($IncludeIrCheck -and -not $isAR)` 在 AR 模式下完全跳过 IR 检查
- **现象：** AR Case 的 meta.json 中没有 irSla 数据，前端 fallback 显示 "unknown"
- **修复：** AR 模式下使用 `$MainCaseNumber` 调用 `check-ir-status.ps1`，将结果写入 AR Case 的 meta 目录，并标注来源为 Main Case

## Acceptance Criteria

- [ ] AR Case 的 IR SLA 显示 Main Case 的 IR 状态（而非 unknown）
- [ ] Meta 中标注 IR 数据来源于 Main Case（如 `irSla._fromMainCase: true`）
- [ ] 非 AR Case 行为不变

## Dependencies
