1. 在 Issues 页面点击某个 issue 的 Create Track 按钮
2. conductor:new-track 被调用，带 ISS-XXX 参数
3. Agent 逐一问：Bug 确认？复现步骤？预期行为？影响范围？测试类型？验证方式？

**实际行为：**
