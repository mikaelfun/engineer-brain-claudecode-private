# Specification: Track Progress Auto-Collapse After Completion

**Track ID:** track-autocollapse_20260321
**Type:** Feature
**Created:** 2026-03-21
**Status:** Draft
**Source:** ISS-036

## Summary

TrackProgressPanel 在 track 创建完成后仍然全部展开，占据空间。完成后应自动折叠为一行紧凑的完成状态，用户可点击展开查看完整状态流历史。

## Acceptance Criteria

- [ ] Track created 后，消息流区域自动折叠，只显示 header 行（✅ Track Created + trackId）
- [ ] Header 右侧有展开/折叠按钮，点击可查看之前的完整状态流记录
- [ ] 创建进行中（isActive）和等待问题（pendingQuestion）时保持展开
- [ ] Error 状态保持展开（需要看到错误信息和 Retry 按钮）
- [ ] TypeScript 编译通过
- [ ] 单元测试通过
