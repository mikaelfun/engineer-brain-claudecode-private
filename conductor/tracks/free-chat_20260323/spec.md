# Specification: AI Assistant Free Chat — 无 session 时允许用户直接聊天

**Track ID:** free-chat_20260323
**Type:** Feature
**Created:** 2026-03-23
**Status:** Draft

## Summary

当前 AI Assistant 面板在没有 active session 时禁用了用户输入框。需要支持 free chat：无 session 时也允许输入，首条消息自动创建新 session。

## Context

AI Assistant 面板设计为必须先执行 step 或 Full Process 才能与 AI 交互。但用户可能想在未执行任何步骤时直接和 AI 聊天（如询问 case 相关问题）。后端 `chatCaseSession` 只支持 resume 已有 session，`POST /case/:id/chat` 在无 session 时返回 404。

## Acceptance Criteria

- [ ] 无 active session 时，chat 输入框和发送按钮可用
- [ ] 用户在无 session 时发送消息，自动创建新 session 并发送
- [ ] 新创建的 session 在 SSE 和 Agent Monitor 中正常显示
- [ ] 有 session 时行为不变（仍然 resume 已有 session）

## Dependencies

- `dashboard/src/agent/case-session-manager.ts` — session 管理核心
- `dashboard/src/routes/case-routes.ts` — chat API 端点
- `dashboard/web/src/components/CaseAIPanel.tsx` — 前端 chat UI

## Out of Scope

- 多 session 并行管理优化
- Chat 历史持久化（已有机制）
