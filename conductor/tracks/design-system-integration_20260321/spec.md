## Summary

将 `playbooks/guides/dashboard-design-system.md` 和 `dashboard/design-preview.html` 中定义的完整双主题设计系统集成到 React 应用中，包括 CSS 变量系统、Tailwind dark mode 配置、主题切换、全组件暗色/亮色适配、布局改造（侧边栏导航）和排版更新。

## Context

Dashboard 是 Engineer Brain 的 Web 管理界面。当前 UI 使用 Tailwind 默认的亮色样式（硬编码 `bg-white`、`bg-gray-50`、`text-gray-900` 等）。已有完整的设计系统文档和独立预览文件，但从未集成到实际 React 应用。

## User Story

As a support engineer, I want the dashboard to have a polished dark/light theme system so that I can work comfortably in different lighting conditions and have a professional, consistent visual experience.
