Watch 卡片和详情面板增加心跳状态判断。

### Tasks

- [x] Task 1.1: 在 AutomationsPage.tsx 中新增 `getWatchHealthStatus()` 工具函数，输入 `(lastPollAt, interval, status, consecutiveErrors)` → 输出 `'running' | 'stale' | 'stopped'`。逻辑：status=stopped → stopped；lastPollAt 为空或 `now - lastPollAt > 3×interval×1000` → stale；否则 → running

- [x] Task 1.4: 在 Watch 详情面板（WatchDetailPanel）中显示 lastPollAt 相对时间和 pollCount 数值
