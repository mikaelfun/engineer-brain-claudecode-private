## Summary

Todo ✅ 仅通知项勾选标记完成后，文本未显示删除线（line-through），完成与未完成项无视觉区分，影响阅读。

## Context

Dashboard Todo 页面（TodoView.tsx）和 Case Detail 的 Todo Tab（CaseDetail.tsx）两处渲染 Todo 项。当前代码在 `section.type === 'green'` 或 `item.checked` 时应用 `line-through`，但用户反馈实际勾选后视觉未更新。需排查是 CSS 渲染问题、状态更新问题、还是条件判断遗漏。

1. 打开 Dashboard → Todo 页面（或 Case Detail → Todo Tab）
2. 找到 ✅ 仅通知 section 的项目
3. 点击勾选标记完成
4. 观察文本样式 — 未出现删除线

**Expected:** 勾选后文本应有 `line-through` 删除线 + 颜色变淡
**Actual:** 勾选成功但文本样式无变化
