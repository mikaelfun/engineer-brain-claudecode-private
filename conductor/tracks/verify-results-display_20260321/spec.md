# Specification: Verify Results Persistent Display

**Track ID:** verify-results-display_20260321
**Type:** Feature
**Created:** 2026-03-21
**Status:** Draft
**Source:** ISS-039

## Summary

执行过 Verify 的 Issue 应该可以在展开详情时看到测试结果。当前 verify 结果只在 modal 中一次性显示，刷新后即丢失。需要：1) 后端将 verify 结果持久化到 issue JSON；2) 前端在 issue 展开详情中显示可折叠的测试结果面板。

## Acceptance Criteria

- [ ] 后端 verify 接口执行后将 `verifyResult` 写入 issue JSON（含 unitTest/uiTest 的 success + output）
- [ ] Issue 展开详情面板中，有 verifyResult 的 issue 显示 "Test Results" 可折叠区域
- [ ] 折叠区域显示各测试步骤的 PASS/FAIL/SKIPPED 状态 + 可展开的 output
- [ ] 刷新页面后测试结果仍可查看
- [ ] TypeScript 编译通过
- [ ] 现有测试通过

## Dependencies

- `issues.ts` 后端路由
- `Issues.tsx` 前端页面

## Out of Scope

- 移除现有的 verify result modal（保留作为执行后即时反馈）
