# Specification: Teams Search Customer Name Matching

**Track ID:** teams-name-match_20260320
**Type:** Feature
**Created:** 2026-03-20
**Status:** Draft

## Summary

Teams 搜索使用 D365 Contact Name 搜索不够精准（如"伟 毕"搜不到 "Bi, Weiwei"）。需要实现中英文客户名映射或模糊匹配机制。

## Context

Teams 搜索 agent 使用 case-info.md 中的 Primary Contact Name 作为搜索关键词。但 D365 存储的联系人名称格式（中文名/拼音/姓在前）与 Teams 中的显示名称（英文名/名在前）经常不匹配。

## Acceptance Criteria

- [ ] 支持中文名→英文名的映射查找（如 "伟 毕" → "Bi, Weiwei" → "Weiwei Bi"）
- [ ] 搜索失败时自动尝试名字顺序反转（"姓 名" → "名 姓"）
- [ ] 可维护一个客户名称映射表（手动补充已知的映射关系）
- [ ] 搜索成功率明显提升

## Dependencies

None

## Out of Scope

- 自动翻译中文名到拼音
- 修改 D365 中的联系人名称

## Technical Notes

可能的方案：
1. 本地映射表 `data/contact-name-map.json`，手动积累
2. 搜索时自动尝试多种格式：原名、反转、去空格、拼音变体
3. Teams 搜索结果缓存，下次直接用上次成功的名称
