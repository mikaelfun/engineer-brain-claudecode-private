
## ISS-226 Teams 图片下载 — Auth 方案验证成功

### 验证成功的方案
1. **MCP Playwright 打开 teams.microsoft.com** → SSO 自动登录（MCP profile 持久化）
2. **从 localStorage 提取 ic3.teams.office.com token**：`localStorage` 中 MSAL cache，`credentialType=AccessToken` + `target` 包含 `Teams.Access`
3. **base64 解码 hostedContents contentId**：`base64.b64decode(contentId)` → `url=https://us-api.asm.skype.com/v1/objects/{objectId}/views/imgo`
4. **Bearer {ic3_token} + Skype API URL** → curl 下载 → 200 OK

### 关键发现
- Graph API hostedContents URL 需要 Chat.Read scope → az CLI 没有 → 403
- 但 contentId 的 base64 里嵌入了 Skype API URL（`us-api.asm.skype.com`）
- Teams web 的 ic3.teams.office.com token（`Teams.AccessAsUser.All` scope）可以访问 Skype API
- Token ~1h TTL，Teams web 自动刷新
